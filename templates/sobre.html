Sobre o projeto

O site Ensinar C foi desenvolvido como parte prática de um Trabalho de Conclusão, com o objetivo de auxiliar estudantes no aprendizado inicial da linguagem de programação C.

Objetivo

O objetivo da plataforma é organizar o estudo da linguagem C em módulos progressivos, permitindo que o aluno tenha contato com explicações curtas, exemplos de código e exercícios práticos.

Metodologia aplicada

A metodologia do site é baseada em uma trilha de aprendizagem. Os conteúdos começam por conceitos básicos, como estrutura de um programa, printf, scanf e variáveis, avançando para operadores, decisões e repetições.

Tecnologias utilizadas

  • HTML para estrutura das páginas;
  • CSS para estilização da interface;
  • JavaScript para interatividade, progresso e desafios;
  • Python com Flask para o back-end;
  • GitHub para versionamento;
  • Render para hospedagem online.

Melhorias futuras

  • Integração de compilador real de C;
  • Sistema de login;
  • Banco de dados para salvar o progresso;
  • Mais módulos avançados;
  • Mini projetos práticos ao final dos módulos.
